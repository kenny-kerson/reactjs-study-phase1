- 스터디 docs : http://bit.ly/2EhFTYJ

## 브랜치
`master`
- 현재 개발중인 최신 소스코드

`init`
- 최초 프로젝트 생성 상태 : create-react-app 으로 프로젝트 생성 상태

`chxx-xxx`
- 챕터별 소스코드 브랜치치
- ex) cho1-environment-setting : 챕터1에서 환경설정 관련 소스코드 브랜

## 스크립트
- react-scripts 사용

`yarn start`
- 개발서버 기동

`yarn build`
- 소스코드 빌드 : transpile + bundling

`yarn eject`
- 소스코드 패키징 풀기 : ex) webpack 설정
